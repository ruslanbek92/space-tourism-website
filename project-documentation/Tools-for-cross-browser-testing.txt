1.Gulp task runner for automating testing tools 
2.gulp-htmltidy for html linting and formatting
3.gulp-csslint for linting css
4.gulp-jshint
5.gulp-autoprefixer  to scan our CSS and add vendor prefixes only where needed
6.gulp-babel to transpile any new JavaScript syntax features to traditional syntax that works in older browsers
